## 🔍 如果功能無法使用

### 問題 1：前端無法連接後端

**症狀**：
- 登入失敗
- 上傳照片失敗
- AI 聊天無法使用

**解決方法**：
1. 確認 GitHub Secret `VITE_API_URL` 已設置
2. 確認 URL 後面有 `/api`
3. 重新運行 GitHub Actions 工作流

### 問題 2：AI 功能無法使用

**症狀**：
- AI 分析返回錯誤
- AI 聊天無法回應

**解決方法**：
1. 確認 Railway 環境變數 `API_KEY` 已設置
2. 確認 Google Gemini API Key 有效
3. 檢查 Railway 日誌是否有錯誤

### 問題 3：後端無法訪問

**症狀**：
- 打開 Railway URL 顯示錯誤
- 前端無法連接後端

**解決方法**：
1. 檢查 Railway 服務狀態（應該是 "Active"）
2. 檢查 Railway 日誌是否有錯誤
3. 確認端口配置正確（8080）
